Flagship · Final Year Thesis
Multimodal Breast Cancer Detection
PyTorch • MedCLIP • TabNet • Early Fusion • Landmark University 2024–2025

Addressed single-modality screening limitations by designing a fusion architecture combining ultrasound imaging with clinical and radiometric data. 87.1% ROC-AUC.

87.1% ROC-AUC 82.7% Accuracy 576-dim Fusion Vector Outperforms Unimodal Baselines
  • MedCLIP (Vision): 512-dim semantic embeddings from ultrasound images via a vision-language model pretrained on medical data.
  • TabNet (Tabular): Attention-based processing of clinical metadata — patient age, BI-RADS, tissue density, radiometric features.
  • Early Fusion: Concatenated 576-dim vector passed through a classification head.
  • Results: 82.7% Accuracy • 87.1% ROC-AUC — significantly outperforming unimodal baselines.
class CancerFusionModel(nn.Module): def __init__(self): self.medclip = MedCLIPModel() # 512-dim self.tabnet = TabNetEncoder() # 64-dim self.head = nn.Linear(576, 2) def forward(self, image, tabular): img = self.medclip(image) # [B, 512] tab = self.tabnet(tabular) # [B, 64] return self.head(torch.cat([img, tab], dim=1))
