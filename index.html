Multimodal Breast Cancer Detection
PyTorch • MedCLIP • TabNet • Early Fusion

B.Sc. Final Year Thesis. Fusion architecture combining ultrasound image embeddings with clinical and radiometric data — 87.1% ROC-AUC.

87.1% ROC-AUC 82.7% Accuracy 576-dim Fusion Vector Outperforms Unimodal

Research Scope: Addressed the core limitation of single-modality screening by designing a fusion architecture combining heterogeneous data sources.

  • MedCLIP (Vision): 512-dim semantic embeddings from ultrasound images via a medical vision-language model.
  • TabNet (Tabular): Attention-based processing of clinical metadata — age, BI-RADS score, tissue density, radiometric features.
  • Early Fusion: Concatenated 576-dim vector passed through a fully connected classification head.
class CancerFusionModel(nn.Module): def __init__(self): self.medclip = MedCLIPModel() # 512-dim self.tabnet = TabNetEncoder() # 64-dim self.head = nn.Linear(576, 2) def forward(self, image, tabular): img = self.medclip(image) tab = self.tabnet(tabular) return self.head(torch.cat([img, tab], dim=1))
